Group exponential.

This functions takes in an element of tangent space (= rotation angle plus logarithm of scale) and returns the corresponding element of the group :ref:`RxSO2 <doxid-class_sophus_1_1_rx_s_o2>`.

To be more specific, this function computes ``expmat(hat(theta))`` with ``expmat(.)`` being the matrix exponential and ``hat(.)`` being the :ref:`hat() <doxid-class_sophus_1_1_rx_s_o2_1a2f19fefd9d78fc745fa996287eece574>` -operator of RSO2.

	static :ref:`Transformation<doxid-class_sophus_1_1_rx_s_o2_1a49e2f23bce49e6d458457c58847df0ce>` hat(:ref:`Tangent<doxid-class_sophus_1_1_rx_s_o2_1ac47f8c66228ef8a3138aec3e9fa3f3c7>` const& a)

hat-operator

It takes in the 2-vector representation ``a`` (= rotation angle plus logarithm of scale) and returns the corresponding matrix representation of Lie algebra element.

Formally, the :ref:`hat() <doxid-class_sophus_1_1_rx_s_o2_1a2f19fefd9d78fc745fa996287eece574>` -operator of :ref:`RxSO2 <doxid-class_sophus_1_1_rx_s_o2>` is defined as

``hat(.): R^2 -> R^{2x2}, hat(a) = sum_i a_i * G_i`` (for i=0,1,2)

with ``G_i`` being the ith infinitesimal generator of :ref:`RxSO2 <doxid-class_sophus_1_1_rx_s_o2>`.

The corresponding inverse is the :ref:`vee() <doxid-class_sophus_1_1_rx_s_o2_1ac07da6e0f4fad63fb460158e8760d9c3>` -operator, see below.

.. index:: pair: function; lieBracket
.. _doxid-class_sophus_1_1_rx_s_o2_1a8abbfb14075fb3de094b6bce9553a8ba:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	static :ref:`Tangent<doxid-class_sophus_1_1_rx_s_o2_1ac47f8c66228ef8a3138aec3e9fa3f3c7>` lieBracket(:ref:`Tangent<doxid-class_sophus_1_1_rx_s_o2_1ac47f8c66228ef8a3138aec3e9fa3f3c7>` const&, :ref:`Tangent<doxid-class_sophus_1_1_rx_s_o2_1ac47f8c66228ef8a3138aec3e9fa3f3c7>` const&)

Lie bracket.

It computes the Lie bracket of RxSO(2). To be more specific, it computes

``[omega_1, omega_2]_rxso2 := vee([hat(omega_1), hat(omega_2)])``

with ``[A,B] := AB-BA`` being the matrix commutator, ``hat(.)`` the :ref:`hat() <doxid-class_sophus_1_1_rx_s_o2_1a2f19fefd9d78fc745fa996287eece574>` -operator and ``vee(.)`` the :ref:`vee() <doxid-class_sophus_1_1_rx_s_o2_1ac07da6e0f4fad63fb460158e8760d9c3>` -operator of :ref:`RxSO2 <doxid-class_sophus_1_1_rx_s_o2>`.

Precondition: ``Omega`` must have the following structure:

.. code-block:: cpp

	|  d -x |
	|  x  d |
